# BuildGradleProject-

How to create gradle project
1.	Make sure you have gradle installed and set in your PATH variable.
2.	Download the Jar and click on it to run.
3.	Give project name and gradle version.
4.	And click on create project.
5.	Project will be created on your desktop.
6.	Add your dependencies into build.gradle. 
7.	To convert your pom.xml dependencies to build.gradle dependencies see project under link https://github.com/kakadeajinkya05/GradleDependencies
8.	Go to that folder and run ‘gradlew eclipse dependencies’ into command prompt to import project into eclipse and download dependencies
